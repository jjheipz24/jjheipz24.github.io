Challenges

Working on this project, I actually encountered less challenges than I expected. However, there were two aspects that were particular tricky. The first was getting the hamburger navigation to appear and work when the window hit a certain size. I spent a long time figuring out how to get it to work, only to realize (from the eyes of Henry) that I made a simple typo, which kept it from working properly. Before Henry looked at it, I was trying to look at W3Schools to see if it could help me.

The second I faced was making a responsive form. I spent far too long going about setting up the form in a way that would make it responsive (like using flexbox), but nothing I did seemed to work. Even now, my contact form is not responsive, and I'm out of time to fix it (but I will go back and fix it when I have more time).

Above and Beyond

For this site, one thing I did that I believe was above and beyond was the implementation of JQuery. Even though it was a very simple usage of it to make the downward arrow do a smooth scroll and create a hamburger dropdown menu effect, it was not required for this project in general. I originally had no idea how I was going to accomplish either. After getting an extension on this project and learning about JQuery in class, I decided I should use this knowledged I gained in the extra time I got to add some extra "bells and whistles" to my site.
